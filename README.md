# Streamlit Cheat Sheet Clone

A modern, interactive clone of the official [Streamlit Cheat Sheet](https://docs.streamlit.io/develop/quick-reference/cheat-sheet), built entirely in Python using Streamlit.

## Features

- **Dual Navigation Modes**: Switch between **Sidebar Navigation** and **Tabs** to suit your viewing preference.
- **Complete Reference**: Covers Install, Magic Commands, Text, Data, Charts, Widgets, Layouts, and more.
- **Premium UI**: Enhanced with custom CSS, animations, and a clean, responsive design.
- **Copy-Paste Ready**: All code snippets are easy to read and copy.

## Local Development

1.  **Clone the repository**:
    ```bash
    https://github.com/onesolution9983/streamlit-cheat-sheet.git
    cd streamlit-cheat-sheet
    ```

2.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the app**:
    ```bash
    streamlit run cheat_sheet.py
    ```

## Deployment on Streamlit Cloud

1.  Push this code to a public GitHub repository.
2.  Go to [Streamlit Cloud](https://streamlit.io/cloud).
3.  Click **New app**.
4.  Select your repository, branch, and set the "Main file path" to `cheat_sheet.py`.
5.  Click **Deploy!**
